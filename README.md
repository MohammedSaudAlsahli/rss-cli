### Install

Simply run:
`pip install rss-cli`
Or
`pip install git+https://github.com/MohammedSaudAlsahli/rss-cli`

### Options

| Options          | Description    |
| ---------------- | -------------- |
| (No option)      | Run the app    |
| `-a`, `--add`    | Add a link     |
| `-r`, `--remove` | Remove a link  |
| `-l`, `--list`   | Show all links |
